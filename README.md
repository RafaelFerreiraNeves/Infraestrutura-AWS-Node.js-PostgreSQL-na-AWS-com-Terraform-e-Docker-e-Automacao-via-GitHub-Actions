# Infraestrutura AWS com Terraform e CI/CD

Projeto de provisionamento de infraestrutura na AWS utilizando Terraform com arquitetura modular, incluindo EC2, RDS e VPC. Implementação de pipeline CI/CD com GitHub Actions e aplicação containerizada com Docker.

## Tecnologias
- AWS (EC2, RDS, VPC)
- Terraform
- Docker
- GitHub Actions

## Arquitetura
- VPC com subnets públicas e privadas
- EC2 rodando aplicação
- RDS PostgreSQL
